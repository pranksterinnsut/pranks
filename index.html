Best Pranks For Computer Lab
1) Fake Desktop

In this prank, we create a fake desktop using screenshot of desktop and browser in full screen mode. When someone click anywhere on the desktop they are directed to a new web site.

Step 1 - Take a screen shot of desktop and name it as img.png.
Step 2 - Download the required HTML file.
Step 3 - Keep both image and file in same folder.
Step 4 - Open the file in browser and keep it on full screen.
Step 5 - Put the device on sleep and wait for some one to be pranked
